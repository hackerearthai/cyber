## 📌 What is Mailed It?

**Mailed It** is an Chrome extension and web platform that protects you from phishing attacks in real time. It scans every email, link, and website you visit — flagging threats in milliseconds using advanced AI models trained on millions of phishing patterns.

Phishing doesn't stand a chance.

## ✨ Features

### 🔍 Live URL Scanner (Web)
- Paste any suspicious URL directly on the website
- Instantly scanned across **35 vulnerability checks** including:
  - SQL Injection
  - XSS (Cross-Site Scripting)
  - Fake domains & typosquatting
  - Phishing keywords
  - Suspicious TLDs
  - IP address URLs
  - URL shorteners
  - Data theft patterns
  - High URL entropy
  - Excessive subdomains
  - Brand name in subdomain
  - Number-for-letter substitution
  - @ symbol tricks
  - Hex/encoded characters
  - Suspicious file extensions
  - Free hosting platform abuse
  - Email tracking service abuse
  - Auto-generated domains
  - Nested URLs in parameters
  - Click tracking paths
  - SendGrid redirect abuse
  - And many more...
  - Get a **threat score out of 100** with a visual riskometer
  - See exactly which checks passed and which failed
  - Filter results by threats found vs safe checks

### 🧩 Chrome Extension Features

#### 🖱️ Link Hover Scanner

- Hover over any URL on any webpage
- The extension instantly shows whether the link is **safe or malicious**
- Non-intrusive popup with threat score and risk indicators
- No clicking required — protection happens on hover

#### 📧 Phishing Email Detection

- Integrates with **Gmail** to scan incoming emails
- Automatically detects phishing emails using AI
- Flags suspicious sender patterns, spoofed domains, and deceptive subject lines
- If a phishing email is detected, the extension **moves it directly to trash** — preventing you from ever interacting with it
- All actions (move to trash / restore) require **user confirmation**

#### 🌐 Real-Time Website Protection

- Every page load is analyzed in **under 80ms**
- If a phishing site is detected, shows a **"Phishing Site Detected"** warning with:
  - Domain spoofing score
  - Credential harvesting score
  - Fake SSL badge detection
  - Intelligent Email Scanning
  - Flags suspicious emails in Gmail & Outlook


## ⚙️ How It Works

#### 01 Extension Activates
   → Mailed It loads silently when Chrome starts
   → No setup required — monitors every tab automatically

#### 02 AI Scans in Real Time
   → Every URL, email, and page is analyzed instantly

#### 03 Threat Score Generated
   → AI produces a score with specific risk indicators

#### 04 You're Protected
   → Threats are blocked, flagged, or moved to trash
